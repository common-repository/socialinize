== Description ==

Is a plugin that allows user to login via there social network accounts such as Facebook, Twitter, Dribble, Linkedln and Tumblr.
Has other features.

  - Social Share.
  - Social Follow.
  - Social Login.

*** Plugin setup requires you to register on Socialinize website to get Socialinize API key. 


== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Create socialinize app for your web application 
3. Create twitter app for your web application 
4. Insert your cient ID and secret  socialinize app website
5. Add socialinize API in plugin setting option.


== Frequently Asked Questions ==

1. What does Socialinize do and why do I need it?
Socialinize provides a service with our Wordpress plugin. that lets users to log in to a website with their Facebook, Twitter and other social media Providers. By using Socialinize, you eliminate traditional registration form & process hencing attracting more users.
2. What is the benefit of using Socialinize?
One of the main benefits is that you don't need to download libraries to login, as a result this does not affect the load time of the website. 
3. When a user logins, I don't get there email address. Why?
The reason is that by default when a user registers with Facebook or Twitter, there email will not be shared. The user can change the setting in there social media to allow to share email address.
